# Grafana dashboards

Import these dashboards manually through **Dashboards → New → Import**:

- `overview.json` — exporter availability, host capacity, pipeline health, and current leader overview;
- `postgresql.json` — PostgreSQL member-level metrics and primary switch timeline;
- `kvrocks-sentinel.json` — Kvrocks and Redis Sentinel metrics, replica offset lag, and master switch timeline;
- `pipeline.json` — hot/cold replication and classifier progress.

During import, map the `DS_PROMETHEUS` input to the existing Prometheus
datasource. The dashboards use the `environment`, `cluster`, `tier`,
`instance`, and SQL exporter `target` labels generated by this configuration.
Leader timelines are derived from the 15-second PostgreSQL and Sentinel scrape
series, so a transition is timestamped with scrape-interval precision.
